# US Wildfire Analysis

## Data Requirements

This project requires the **1.88 Million US Wildfires** dataset from Kaggle to run the `.Rmd` file.

Please download the dataset `.sqlite` before running the analysis:

🔗 [1.88 Million US Wildfires – Kaggle Dataset](https://www.kaggle.com/datasets/rtatman/188-million-us-wildfires)

Once downloaded, place the dataset file in the project's working directory before knitting the `.Rmd`.
